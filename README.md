H5P Book Maker
==========

Forked from H5P.CoursePresentation

## Getting started

Install all the dependencies:

```bash
npm install
```

Build project for production:

```bash
npm run build
```

Build continuously

```bash
npm run watch
```

Run tests:

```bash
npm test
```
